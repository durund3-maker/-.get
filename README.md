# 爬虫项目


## 文件概览

* `雪球.py` — 实现对单只个股的**日度行情数据**爬取（雪球）。
* `微博爬取.py` — **按照关键词抓取微博帖子**（文本、作者、时间、转发/评论/点赞数等）。
* `关键词爬取_redbook.zip` — 一个压缩包，**按关键词抓取小红书笔记**的脚本。
* `redbook.py` — **小红书自动化评论脚本**。
* `股吧评论分析.py` — 对股吧评论进行**情感分析**
* `greenwash.py` — 基于异质性代理人的**演化博弈仿真结果可视化**脚本（生成演化轨迹、群体分布热图等）。
* `Failed_2025.9.10 微博与男女平权/` — 一个**爬取案例**目录，包含微博爬取与文本分析。目标是比较中国互联网上男性主义与女性主义话语权，但因样本过少未得出结论。保留以供复现与方法论反思。
